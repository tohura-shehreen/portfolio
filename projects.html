Group Thesis

Neural Machine Translation for Chakma–Bangla

My role: model design & training pipeline

My undergraduate thesis: an exploration of low-resource techniques for translating Chakma — an Indigenous language of the Chittagong Hill Tracts with very little parallel data — into Bangla.

  • Fine-tuned BanglaT5 using iterative back-translation to make the most of a tiny parallel corpus
  • Reached a BLEU score of 19.00 — a 30.4% improvement over baseline
  • Matched GPT-4.1's in-context learning performance and outperformed every prior fine-tuned system on this pair
PythonPyTorchHuggingFaceBanglaT5sacrebleu
View on GitHub ↗
ProblemExtreme low-resource machine translation — very few Chakma-Bangla sentence pairs exist for training.
ApproachIterative back-translation on top of a BanglaT5 base model, cycling synthetic parallel data to bootstrap quality.
ResultBLEU 19.00, +30.4% over baseline; on par with GPT-4.1 in-context learning; best fine-tuned result to date on this language pair.
StatusComplete — submitted as undergraduate thesis, June 2026.

eBird Bias Correction (GNN + XGBoost)

My role: end-to-end — preprocessing through documentation

Citizen-collected bird sighting data is famously biased: some areas get birdwatched constantly, others almost never. This pipeline models and corrects for that observation bias.

  • Built a Graph Neural Network to capture spatial relationships between observation sites
  • Layered XGBoost on top for the final bias-corrected prediction
  • Owned the full pipeline: preprocessing, modeling, evaluation, and documentation

ProblemCitizen-science observation data (eBird) is spatially biased by where people choose to birdwatch.
ApproachGraph Neural Network for spatial structure, feeding into an XGBoost model for the corrected prediction.
ResultA working correction pipeline, fully documented from raw data to evaluation.
StatusComplete.
