Group Thesis

Neural Machine Translation for Chakma-Bangla

My undergraduate thesis: an exploration of low-resource techniques for translating Chakma — an Indigenous language of the Chittagong Hill Tracts with very little parallel data — into Bangla.

  • Fine-tuned BanglaT5 using iterative back-translation to make the most of a tiny parallel corpus
  • Reached a BLEU score of 19.00 — a 30.4% improvement over baseline
  • Matched GPT-4.1's in-context learning performance and outperformed every prior fine-tuned system on this pair
View on GitHub PythonPyTorchHuggingFaceBanglaT5
Problem
Extreme low-resource machine translation — very few Chakma-Bangla sentence pairs exist for training.
Approach
Iterative back-translation on top of a BanglaT5 base model, cycling synthetic parallel data to bootstrap quality.
Result
BLEU 19.00, +30.4% over baseline; on par with GPT-4.1 in-context learning; best fine-tuned result to date on this language pair.
